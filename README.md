# Utilisation Manual
API Link : `https://spacemoonapis.frb.io/webdocressources/{id}`

L'api est utilisable unique avec un `/webdocressources/{id}` listé ci-dessous, si vous tapez `/webdocressources/` vous allez tomber sur une erreur.
